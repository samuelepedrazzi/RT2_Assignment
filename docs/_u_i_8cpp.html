Detailed Description

Final assignment: user interface for a mobile robot using RViz and Gazebo.

Author
Samuele Pedrazzi
Version
1.0
Date
24/04/2022

Services :
/gazebo/reset_simulation used to reset the simulation in gazebo

Description :

This node allows the user to select the robot's movement modes: ° establishing the coordinates of a target to be accomplished, and the robot calculating a path to reach it dynamically ° controlling it via keyboard ° controlling it via keyboard with collision avoidance to avoid obstacles

The robot is navigating through an unknown environment, but owing to laser scanner sensors, it can create a map and recognize barriers.
